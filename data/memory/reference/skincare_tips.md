# 护肤知识

## 基础护肤步骤

### 晨间护肤
1. 洁面 - 清除夜间分泌的油脂
2. 爽肤水 - 二次补水
3. 精华 - 根据需求选择
4. 面霜 - 保湿锁水
5. 防晒 - 必不可少

### 晚间护肤
1. 卸妆 - 彻底清洁彩妆
2. 洁面 - 清洁一天污垢
3. 爽肤水 - 补水
4. 精华 - 修护成分
5. 面霜/乳液 - 夜间修护
6. 眼霜 - 眼周护理

## 肤质类型

- 干性皮肤：容易缺水，需要保湿
- 油性皮肤：油脂分泌旺盛，需要控油
- 混合性皮肤：T区油U区干，需要分区护理
- 敏感性皮肤：容易过敏，需要温和护理
- 中性皮肤：理想状态，维持即可

## 常见护肤成分

- 烟酰胺：美白、控油、修护
- 维C：抗氧化、美白
- 透明质酸：保湿
- 神经酰胺：修护屏障
- 视黄醇(A醇)：抗衰老
- 水杨酸：去角质、控油
- 果酸：去角质、美白
- 壬二酸：祛痘、美白
